# @projects/quest-hunt (QuestHunt - Geocaching Adventure Platform)

## 📱 Overview

QuestHunt is a mobile-first web application that transforms geocaching into an engaging, social, and interactive
experience. It enables users to create, participate in, and share location-based treasure hunts with friends and the
community.

This project has been integrated into the [GameHub Monorepo](../../../README.md).

For development, testing, and deployment instructions, please refer to the main monorepo documentation.

### Commands

- **Build**: `pnpm --filter @projects/quest-hunt build`
- **Lint**: `pnpm --filter @projects/quest-hunt lint`
- **Test**: `pnpm --filter @projects/quest-hunt test`
